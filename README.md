### 📖 项目简介
VideoHub 是一款基于 Flutter 开发的跨平台视频库应用，代码由AI完成编写
支持 Android、iOS、macOS、Linux、Windows 五大平台，提供视频管理和播放功能。
### 🖥️ 平台支持
平台 状态 备注 Android ✅ Android 5.0+ iOS ✅ iOS 9.0+ macOS ✅ macOS 10.9+ Windows ✅ Windows 7+ Linux ✅ 现代发行版

### ✨ 核心功能 🎬 多协议视频源支持
- SMB/CIFS - Windows 共享文件夹、NAS
- FTP/SFTP - 文件传输协议
- WebDAV - Nextcloud、ownCloud、坚果云
- HTTP/HTTPS - 直链视频
- Emby/Jellyfin - 媒体服务器 （TODO）
- TVBox - 盒子源解析 🎥 视频播放器
- 双播放引擎 - FVP (FFmpeg) / MediaKit 可切换
- 平台自适应 - Linux 优先 FVP，其他平台优先 MediaKit
- 播放控制 - 播放/暂停、快进/快退、进度拖动
- 缓存进度显示 - 实时显示视频缓存状态
- 倍速播放 - 多种播放速度选择
- 音量控制 - 手势调节音量
- 全屏播放 - 横竖屏自适应 📂 媒体库管理
- 自动刮削 - TMDB 元数据匹配
- 海报展示 - 瀑布流卡片布局
- 分类管理 - 电影/剧集自动分类
- 收藏功能 - 收藏喜爱的影片（TODO）
- 播放历史 - 自动记录播放进度 🔍 视频搜索
- TVBox 源 - 支持导入自定义源（TODO）
- 关键词搜索 - 快速定位视频 🎨 用户界面
- Material Design 3 - 现代化 UI 设计
- 深色模式 - 护眼夜间主题
- 响应式布局 - 适配手机/平板/桌面
- 多语言支持 - 中文/English



### 🔧 技术栈
- Flutter 3.x / Dart 3.x
- BLoC 状态管理
- go_router 路由
- FVP / MediaKit 播放器
- Hive 本地存储
- TMDB API 刮削
